# Minimalist Icons - Prompt-Patterns

> Kategorie fuer Infografiken, Makro-Fotografie, wissenschaftliche Diagramme und minimalistische Icon-Designs. Nano Banana Pro erzeugt praezise technische Visualisierungen und detailreiche Nahaufnahmen.

## Empfohlenes Format

Kurze, praegnante Prompts fuer Makro- und Icon-Generierung. Fuer wissenschaftliche Diagramme genuegt oft ein einfacher beschreibender Satz.

## Best Practice Patterns

### Pattern 1: Mikroskopische Makro-Visualisierung

Erzeugt hyperdetaillierte Nahaufnahmen alltaeglicher Objekte, als waeren sie unter einem Mikroskop fotografiert. Minimaler Prompt mit maximalem Detailgrad.

```
Imagine a pencil tip seen through a microscope. Imagine how the pencil tip would look and create an image.
```

### Pattern 2: Fantasy Pixel Art Icon-Grid fuer Game UI

Erzeugt einheitliche Pixel-Art-Icon-Sets mit Beschriftung, ideal fuer Spieloberflaechen und App-Design.

```
A vibrant grid of pixel art icons depicting various fantasy spells and abilities, such as Fireball, Heal, Poison, Teleport, and Meteor. Each icon is distinct, detailed in its pixelated style, and labeled with its corresponding ability name, suitable for a video game user interface.
```

### Pattern 3: Wissenschaftliches Multi-Panel-Diagramm

Erzeugt praezise wissenschaftliche Illustrationen mit mehreren Panels und korrekter Beschriftung. Ideal fuer Lehr- und Praesentationsmaterialien.

```
In multi panels create IL-2, IL-4, IL-12, IL-6 signaling pathways
```

## Kategoriespezifische Tipps

- **Makro-Fotografie**: "seen through a microscope" aktiviert hyperdetaillierte Texturen
- **Icon-Grids**: Explizit "grid" und "labeled" angeben fuer einheitliche Anordnung mit Beschriftung
- **Wissenschaftliche Diagramme**: Einfache Beschreibung reicht -- Nano Banana Pro ergaenzt korrekte Details
- **Pixel Art**: "pixelated style" und spezifische Farben/Themen angeben
- **Infografiken**: Notebook-Paper-Hintergrund und Flowchart-Elemente fuer Lehr-Aesthetik
- **Minimalismus**: Weniger Prompt-Text fuehrt oft zu klareren, reduzierteren Ergebnissen

## Empfohlene Negative Prompts

```
cluttered background, overlapping elements, blurry text, inconsistent icon sizes, photorealistic style (when pixel art desired), 3D effects on flat icons, watermark, low resolution, misaligned grid, illegible labels
```
